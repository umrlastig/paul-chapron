Chapitre 20 la Cartographie comme support

20.1 Cartographier le résultats de traitements statistiques

Une carte permet d’explorer “visuellement” des résultats de traitements statistiques :

  • Cartographie univariée : Cartographie électorale
  • Cartographie bi-variée : représenter les résidus d’un modèle explicatif à une variable
  • Cartographie multi-variée : Cartographie de clusters, de coordonnées d’individus dans le plan factoriel (ACP), etc.

Typiquement : les résidus d’un modèle sont-ils homogènes dans l’espace ? Si non , pourquoi ?
